# Fitness-Classification
A classification project using a **Multilayer Perceptron (MLP)** to determine fitness eligibility. This model explores non-linear relationships in health data to categorize individuals as 'Fit' or 'Not Fit'.
##  Model Architecture
* **Hidden Layers:** Dual-layer architecture (16 nodes, 8 nodes).
* **Activation:** ReLU (Rectified Linear Unit).
* **Optimizer:** Adam (Adaptive Moment Estimation).
* **Maximum Iterations:** 1200 (to ensure loss convergence).

##  Key Highlights
* **Preprocessing:** Label encoding for categorical health data and StandardScaling for feature uniformity.
* **Analysis:** Includes a class distribution plot to check for dataset balance.
* **Evaluation:** Comprehensive classification report (Precision, Recall, F1-Score) and Confusion Matrix.

##  Tech Stack
* Python, Scikit-Learn, Pandas, Seaborn,Neural Networks
